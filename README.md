## Quick start

### Last FM API Key

- Request your own LastFM API Key: https://www.last.fm/api/
- And save it as `REACT_APP_LASTFM_API_KEY`in your environment variables (`.env`)

### Install dependencies
In the project directory:

- `npm install`

### Start application

- `npm start`
